# 자바 자료구조
## 자바 프로그램의 기초 

> 기본 데이터 타입

    1) 정수형 데이터
    byte, char, short, int, long
    2) 실수형 데이터
    float, double
    3) 논리형 데이터
    boolean

----
1) 정수형 데이터
    * byte : 8비트
    * char : 16비트
    * short : 16비트
    * int : 32비트 
    * long : 64비트

----
2) 실수형 데이터
    * float : 32비트
    * double : 64비트
----
3) 논리형 데이터(true, false)
    * boolean : 1비트
