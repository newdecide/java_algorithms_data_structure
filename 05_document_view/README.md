# API 보는법
    자바언어를 공부할 때 필수적인 API
> <a href="https://docs.oracle.com/javase/8/docs/api/">자바8 API</a>

* java.lang
    - 자바 데이터 타입(String, Integer, Float 등) 자바언어 사용을 위한 기본적인
     패키지와 클래스 모음

* java.util
    - Collection, Date, Hashmap 등 데이터를 저장하는 클래스들 모음

* java.io
    - 입력과 출력을 제어할 수 있는 클래스 모음

* java.net
    - 네트워크 통신을 위한 기능 모음
